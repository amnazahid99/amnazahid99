## 🚀 About Me

I'm a final-year **Software Engineering** student at **Lahore College for Women University**, building things at the intersection of AI and the web.

Right now I'm deep into **AI/ML/DL & Agentic AI** — LangChain, LangGraph, RAG pipelines, and Ollama. On the creative side, I run **@visualcraftco** — a freelance design page for creating social media graphics, brand identity, and UI/UX projects.

If I'm learning something new, chances are I'm already building a project with it.
